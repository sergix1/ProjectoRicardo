Joc Online de Dispars
Desenvolupat per Sergi Martínez Vayá

Projecte multijugador desenvolupat en Unity 6 utilitzant Unity Netcode for GameObjects.

Característiques
Moviment multijugador
Dispars en xarxa
Sistema de vida sincronitzat
Respawn automàtic
Ragdoll al morir
Botiquins de curació
Controls
WASD → Moure's
Espai → Saltar
Ratolí → Apuntar
Click esquerre → Disparar
Gameplay

Els jugadors poden disparar-se entre si i recollir botiquins per recuperar vida. Quan un jugador mor, reapareix automàticament després d’uns segons.

Estat

Projecte funcional amb multijugador online bàsic implementat.
